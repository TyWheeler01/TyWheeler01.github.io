Enhancement 1: Goal Weight Tracker App

Artifact 1: CS 360 – Mobile Architecture and Programming

Category: Software design and engineering

This Android application allows users to log their weight, set goal weights, and receive notifications according the the milestones and goals they set. The original version of the app, built in Java using SQLite, lacked modern architecture, secure logins, and visual clarity. This enhanced version addresses these issues through a few significant updates.

Enhancements completed:

  • Implemented Firebase Authentication with email verification and simulated two-factor authentication (2FA).
  • Refactored app architecture using MVVM (Model-View-ViewModel) pattern.
  • Redesigned the user interface using Material Design components.
  • Validated weight and date inputs using regex and SimpleDateFormat.
  • Added back button navigation with consistent behavior across all screens.
